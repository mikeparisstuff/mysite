Color Clock

For day two I actually got the idea from another site I had seen and thought it would be a great opportunity to get my feet wet. The whole idea is that we can convert the current time into a 6 byte hexidecimal representation for a color thus creating the color of now. The only interesting part of the code is that setInterval() function which allows you to schedule funciton calls on a rolling basis. This allows us to update the current time and convert that time into a color which automatically becomes the background.


Additionally, I am looking into options for a persistent datastore. I am intrigued with the noSQL stores such as redis, mongoDB, and couchDB which seem to be all the rage these days. After some research, I decided that I am going to give mongoDB a shot and began to implement that logic into the node application today. My goals for the immediate future are to set up a simple datastore to hold these blog posts but it may also allow me some more flexibility in the types of pages I will create going forward. I am still brainstorming for what will come tomorrow so check in soon.

A Site a Day

It is time. It is time that I stop having a "good enough" relationship with html, css, and javascript and start becoming a designer for the web. Why? I have spent the last 3 months working on backend solutions for a company that specializes in creating mobile applications. As a result, I have become increasingly proficient with backend technologies but rarely get to touch the front end code because that is handled, naturally, by the native client teams. My vision for this project and the ones which follow is to improve my skill in web design, as well as to document my journey along the way. I have 28 days before school starts back up, and 28 websites to create before then. Let's get started.
