A Site a Day

It is time. It is time that I stop having a "good enough" relationship with html, css, and javascript and start becoming a designer for the web. Why? I have spent the last 3 months working on backend solutions for a company that specializes in creating mobile applications. As a result, I have become increasingly proficient with backend technologies but rarely get to touch the front end code because that is handled, naturally, by the native client teams. My vision for this project and the ones which follow is to improve my skill in web design, as well as to document my journey along the way. I have 28 days before school starts back up, and 28 websites to create before then. Let's get started.
